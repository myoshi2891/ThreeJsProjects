# Three.js Projects

Three.jsを使用したインタラクティブ3D Webプロジェクト集。

## Projects

| Project | Description | Tech |
|---------|-------------|------|
| [Hope](./Hope) | 希望をテーマにした没入型3Dエクスペリエンス | Three.js, GSAP, TypeScript |

## Quick Start

```bash
# Hopeプロジェクトを起動
cd Hope
bun install
bun dev
```

## Development Environment

- **Runtime**: Bun
- **Build**: Vite
- **Language**: TypeScript
- **3D**: Three.js
